# Geschichte

Um dieses Projekt zu starten befolge diese Schritte:
1. Klicke auf den grünen Button "Code"
2. Klicke auf Download ZIP
3. Extrahiere die ZIP in irgendeinen Ordner
4. Öffne "Main.html"
